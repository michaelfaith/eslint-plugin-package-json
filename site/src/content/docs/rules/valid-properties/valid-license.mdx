{/* end auto-generated rule header */}

import { TabItem, Tabs } from '@astrojs/starlight/components';

This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `license` property, if present, against the following criteria:

- It must be a `string`
- The value should be a valid SPDX license, "UNLICENSED", or a note referencing a license file (e.g. "SEE LICENSE IN LICENSE.md")

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "license": "GPL3"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "license": "GPL-3.0-only"
}
```

</TabItem>
</Tabs>
