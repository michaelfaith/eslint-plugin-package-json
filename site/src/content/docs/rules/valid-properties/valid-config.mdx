import { TabItem, Tabs } from '@astrojs/starlight/components';

The rule checks that, if present, the `config` property is an object.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "config": "config.json"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "config": {
    "name": "foo",
    "port": "8080"
  }
}
```

</TabItem>
</Tabs>
