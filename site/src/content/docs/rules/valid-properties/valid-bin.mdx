This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `bin` property, if present, against the following criteria:

- It must be either a non-empty string or an object.
- If the value is an object, it should only consist of non-empty string values.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "bin": {
    "invalid-bin": 123
  }
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "bin": "./bin/cli.js"
}
```

```json
{
  "bin": {
    "my-cli": "./bin/cli.js"
  }
}
```

</TabItem>
</Tabs>
