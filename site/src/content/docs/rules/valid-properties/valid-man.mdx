import { TabItem, Tabs } from '@astrojs/starlight/components';

This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `man` property, if present, against the following criteria:

- It must be a string or an Array of strings
- The string(s) must end with a number (and optionally .gz)

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "man": ["./man/doc.md"]
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "man": ["./man/doc.1", "./man/doc.2"]
}
```

```json
{
  "man": "./man/doc.1"
}
```

</TabItem>
</Tabs>
