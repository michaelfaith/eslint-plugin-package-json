This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `name` property, if present, against the following criteria:

- It must be a string
- It should pass [`validate-npm-package-name`](https://www.npmjs.com/package/validate-npm-package-name) validation

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "name": "Exciting! Package! Name!"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "name": "exciting-package-name"
}
```

</TabItem>
</Tabs>
