This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `os` property, if present, against the following criteria:

- It must be an Array
- All items in the Array should be one of the following: "aix", "android", "darwin", "freebsd", "linux", "openbsd", "sunos", and "win32"

<Aside>
  These values are the list of possible `process.platform` values [documented by
  Node](https://nodejs.org/api/process.html#processplatform).
</Aside>

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "os": ["x64"]
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "os": ["linux", "win32"]
}
```

</TabItem>
</Tabs>
