This rule checks that the `gypfile` property, if present, is a boolean.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "gypfile": "false"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "gypfile": false
}
```

</TabItem>
</Tabs>
