This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `contributors` property, if present, against the following criteria:

- It must be an array of objects.
- Each object should have at least a `name`, and optionally `email` and `url`.
- `email`, if present, should be valid email address.
- `url`, if present, should be a valid URL.

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "contributors": "Trent Reznor"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "contributors": [
    {
      "name": "Trent Reznor",
      "email": "treznor@nin.com",
      "url": "https://nin.com"
    }
  ]
}
```

</TabItem>
</Tabs>
