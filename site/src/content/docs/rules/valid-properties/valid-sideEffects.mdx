💼 This rule is enabled in the following configs: ✅ `recommended`, 📦 `recommended-publishable`.

{/* end auto-generated rule header */}

import { TabItem, Tabs } from '@astrojs/starlight/components';

This rule uses [`package-json-validator`](https://github.com/michaelfaith/package-json-validator) to validate the `sideEffects` property, if present, against the following criteria:

- It must be either a boolean or an Array
- If the value is an Array, it should only consist of non-empty strings

## Examples

<Tabs>
<TabItem label="❌ Incorrect">

```json
{
  "sideEffects": "false"
}
```

</TabItem>
<TabItem label="✅ Correct">

```json
{
  "sideEffects": false
}
```

```json
{
  "sideEffects": ["./dist/polyfills.js"]
}
```

</TabItem>
</Tabs>
