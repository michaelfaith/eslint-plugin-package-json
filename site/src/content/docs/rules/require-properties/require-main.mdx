import { Aside, Tabs, TabItem } from '@astrojs/starlight/components';

This rule checks for the existence of the `main` property and reports a violation if it doesn't exist.

<Aside>
  The `main` property is generally not needed anymore, if you're using a modern
  bundler that supports the `exports` field. Consider using the
  [`require-exports`](/rules/require-properties/require-exports) rule instead,
  which requires that the `exports` field is present.
</Aside>

## Examples

</TabItem>
<TabItem label="✅ Correct">
